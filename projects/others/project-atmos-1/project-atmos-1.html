Project Atmos

Status In progress
Category Electronics / Maker
Type Personal build

Building a custom all-in-one weather station from scratch.

What I think about the project

We started out with a simple weather station. Now something much more interesting is taking shape.

My vision of what it could be able to do:

  • 🌡️ Temperature
  • 💧 Humidity
  • 🌡️ Pressure
  • 🫧 CO₂
  • 🌫️ PM1 / PM2.5 / PM4 / PM10
  • 📊 VOC
  • 🚗 NOx
  • 🟠 UV
  • 💡 Light intensity
  • 🔋 Battery status
  • 🟡 Solar panel output
  • 🕐 Clock
  • 📈 Measurement history
  • 🖥️ Web interface
